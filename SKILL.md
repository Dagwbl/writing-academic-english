# Academic English Writing

## Core Approach

Use the Oshima-Hogue progression: writing process -> paragraph structure -> unity and coherence -> support -> essay structure -> organization patterns -> source use -> sentence structure -> editing.

Treat academic writing as controlled meaning design, not decoration. Diagnose the argument and organization before polishing sentences.

For any substantial revision, coaching, or AI-writing prompt, read `references/academic-writing-framework.md` and apply its checklists. For a short line edit, use the quick workflow below.

## Quick Workflow

1. **Classify the task**: paragraph, essay, source-use task, sentence-level edit, or AI prompt.
2. **Find the controlling idea**: topic sentence for a paragraph; thesis statement for an essay; position claim for an argument.
3. **Check development**: each paragraph needs one main idea, sufficient specific support, and analysis that explains the support.
4. **Check coherence**: repeat key terms when useful, keep pronouns consistent, use transition signals accurately, and arrange ideas in a logical order.
5. **Check genre pattern**: choose logical division, chronological/process, cause/effect, comparison/contrast, or argumentative organization.
6. **Check source discipline**: distinguish facts from opinions, integrate quotations/paraphrases/summaries, cite available sources, and never fabricate sources.
7. **Edit sentences last**: repair fragments, run-ons, comma splices, choppy/stringy sentences, parallelism, clause use, punctuation, and word choice.

## Non-Negotiables

- Preserve the author's meaning, data, stance, citation intent, and disciplinary terminology unless the user asks for a developmental rewrite.
- Do not invent sources, findings, page numbers, methods, statistics, quotations, or references.
- Do not reproduce textbook passages, exercises, or examples. Use original explanations and examples.
- Prefer clear, precise academic English over ornate language.
- Separate global issues from local issues. If the thesis, support, or organization is weak, say so before giving a surface polish.
- For Chinese-language requests, explain feedback in Chinese and provide revised academic prose in English unless the user asks otherwise.

## Task Routing

### Polish a Draft

Return:

1. **Revised version**: clean academic English that preserves meaning.
2. **Key changes**: 3-6 notes on thesis/topic focus, paragraph unity, coherence, evidence, tone, or sentence repair.
3. **Open issues**: only if support, citations, facts, scope, or terminology need confirmation.

Use light edits when the structure works. Use developmental edits when the paragraph or essay lacks a controlling idea, unity, sufficient support, or a defensible organization pattern.

### Diagnose Writing

Use this order:

1. Topic sentence or thesis statement
2. Unity and paragraph role
3. Supporting details and evidence
4. Coherence and transition logic
5. Essay pattern and paragraph order
6. Source integration and citation discipline
7. Sentence structure, punctuation, and style

Lead with the few changes that will most improve the paper. Do not bury major argument problems under grammar comments.

### Guide AI Academic Writing

When writing instructions for another AI, specify:

- genre and audience
- thesis, topic sentence, or position claim
- required organization pattern
- paragraph roles and evidence rules
- source constraints and no-fabrication rule
- tone: formal, precise, concise, appropriately hedged
- output format: revised draft, feedback, outline, paragraph plan, or teaching explanation

### Teach a Pattern

Use a compact lesson shape:

1. Name the pattern.
2. Explain the rule in one or two sentences.
3. Show an original corrected example.
4. Give a reusable template.
5. Ask the user to apply it only if practice is requested.

## Revision Priority

When tradeoffs arise, use this hierarchy:

1. Meaning and factual integrity
2. Controlling idea
3. Unity and support
4. Coherence and organization pattern
5. Source use and documentation
6. Sentence structure and punctuation
7. Academic tone, concision, and style

Do not make prose sound more sophisticated at the cost of accuracy, clarity, or argumentative control.
