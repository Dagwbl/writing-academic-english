# Academic English Writing

A Codex skill for polishing, diagnosing, drafting, and coaching academic English. It applies a paragraph-to-essay workflow informed by Oshima and Hogue: controlling ideas, unity, coherence, support, organization, source discipline, and sentence-level editing.

## Use

Invoke the skill for academic paragraphs, essays, research writing, source integration, thesis statements, ESL/EFL feedback, or prompts for scholarly prose:

```text
Use $academic-english-writing to revise this draft.
```

For substantial revisions, the skill uses the reference framework to address global issues before sentence-level polish.

## Contents

- `SKILL.md`: skill instructions and task-routing guidance.
- `references/academic-writing-framework.md`: detailed revision and teaching framework.
- `agents/openai.yaml`: Codex interface metadata.

## Principles

- Preserve the writer's meaning, terminology, data, and citation intent.
- Do not invent sources, evidence, quotations, or bibliographic details.
- Prioritize argument, organization, and evidence before grammar and style.
- Prefer clear, precise academic English over inflated prose.
